# Bash DBMS Project

A simple command-line Database Management System implemented using shell scripting.  
The project provides a menu-driven interface to manage databases and tables, insert and view records, and delete data.

## Features
- Create, list, connect, and drop databases
- Create tables with a primary key
- Insert records with validation
- Display table contents
- Delete records by primary key

## How to Run
```bash
chmod +x dbms.sh
./dbms.sh
```
